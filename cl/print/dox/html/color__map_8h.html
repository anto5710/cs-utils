Detailed Description

A enum-like container of ANSISequence constants, each being a labled ANSI code string that can be easily appended to output stream.

Author
Dohyun Ahn (dohyu.nosp@m.n.ah.nosp@m.n@tuf.nosp@m.ts.e.nosp@m.du) Each ANSI enum is created with ANSI code, a style name, and a regular expression to detect the code.

e.g.) For black foreground (font) sequence:

C_BLACK("30", "Foreground:Black", "(c:)?bla?ck");

It has ANSI code 30, and should be (will to string and) printed as , resulting Black font color. Its tag is in pattern (c:black) or (black), or shorter.
